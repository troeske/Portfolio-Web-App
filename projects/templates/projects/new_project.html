DESIGN PHILOSOPHY

A project is build up of general project information that includes a description, an image for the project list and one for the details page as well as project timeframe. Among other things the general information includes a flag if the project is confidential and cannot be shown without approved registration.

To provide the ability to structure the project content you can define any number of SECTIONS within the project. Each section includes optionally up to two headings, text content and the display order in which the secions should be displayed.

Each section in turn can include any number of IMAGES, VIDEOS and external LINKS (e.g. to your Github repo)

Each project can optionally also include a section for CATEGORIES that may apply to a project e.g. like "full-stack development", "app-development", "UI/UX Design".

Finally, each project can optionally include a section for LEARNINGS that you took away from the project.

Follow the steps outlined below to create a new project.

STEP 1 - PROVIDE THE PROJECT FRAME

Click on the link below to the site admin panel and fill in the general information about the project. Select your name in the [Consultant id] drop-down:

PROJECT

STEP 2 (optional) - CREATE SECTIONS

Click on the link below to create any section for the project. Select the project you just created in the [Project id:] dropdown:

SECTIONS

STEP 3 (optional) - ADD IMAGES TO A SECTION

Click on the link below to add images to a section of the project. Select the section you want to add images to in the [Section:] dropdown:

SECTION IMAGES

STEP 4 (optional) - ADD VIDEOS TO A SECTION

Click on the link below to add videos to a section of the project. Select the section you want to add videos to in the [Section:] dropdown:

SECTION VIDEOS

STEP 5 (optional) - ADD LEARNINGS TO A SECTION

Click on the link below to add learnings to the project. Select the project you just created in the [Project id:] dropdown:

LEARNINGS
